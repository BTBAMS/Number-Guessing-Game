# 🎲 Number Guessing Game

A fun command-line game built with Python where the computer picks
a secret number and you have to guess it!

## 🎮 How to Play
1. Run the script in your terminal
2. Enter your name
3. Guess a number between 1 and 100
4. Use the hints (Too High / Too Low) to find the secret number
5. Try to guess it in as few attempts as possible!

## ⭐ Rating System
| Attempts | Rating |
|----------|--------|
| 1 - 5    | 🏆 Amazing! |
| 6 - 8    | ⭐ Great job! |
| 9 - 12   | 👍 Not bad! |
| 13+      | 😅 Keep going! |

## 🛠️ Features
- Input validation (handles letters, symbols, out-of-range numbers)
- Play again option
- Personalized messages
- Attempt-based rating system

## 🚀 How to Run
```bash
python number_guessing_game.py
```
